# 装饰板知识图谱可视化应用

## 项目简介

本项目是一个基于Web的交互式“装饰板知识图谱”可视化应用。它旨在通过图形化的方式展示不同装饰板材的性能、特点等信息，并提供详细的查看功能，帮助用户更好地理解和比较各种板材。

## 功能特性

- **交互式知识图谱**: 使用Apache ECharts库将板材数据以树状结构进行可视化展示，用户可以方便地浏览不同板材之间的关系。
- **详细信息展示**: 点击图谱中的节点，侧边面板会显示该板材的详细信息，包括名称、类型、特性描述等，并配有对应的图片。
- **主题切换**: 提供主题切换功能，提升用户体验。

## 技术栈

- **前端**:
    - HTML5
    - CSS3
    - JavaScript
    - Apache ECharts (数据可视化库)

## 项目结构

- `index.html`: 应用程序的入口文件，定义了页面的整体结构，包括ECharts图谱容器和详细信息展示面板，并引入了相关的JavaScript文件。
- `app.js`: 核心应用程序逻辑文件。负责初始化ECharts实例，将原始数据转换为图谱所需的树状结构，渲染图表，处理用户点击事件以显示详细信息，以及管理主题切换。
- `board_data.js`: 应用程序的数据源文件。包含一个名为 `BOARD_DATA` 的常量，它是一个对象数组，每个对象都描述了一种装饰板材及其所有属性。
- `data/`: 该目录用于存放不同板材材料的图片，这些图片在详细信息面板中展示。

## 如何运行

1.  克隆或下载本项目到本地。
2.  使用任意现代浏览器打开 `index.html` 文件即可。无需后端服务器或复杂的配置。
